# Data Directory

This directory should contain the NFL Big Data Bowl 2023 dataset files.

## Required Files

Place the following CSV files in this directory:

- `plays.csv` - Play-by-play data
- `games.csv` - Game metadata
- `players.csv` - Player information
- `week1.csv` through `week8.csv` - Player tracking data for each week
- `pffScoutingData.csv` - PFF scouting data (optional)

## Download

Download the dataset from:
- **Kaggle**: https://www.kaggle.com/competitions/nfl-big-data-bowl-2023/data
- Or search for "NFL Big Data Bowl 2023" on Kaggle

## Notes

- These files are excluded from git due to their large size (~100+ MB total)
- The tracking data files (week*.csv) are particularly large
- You'll need a Kaggle account to download the data
